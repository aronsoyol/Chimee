1. ᠴᠠᠭᠠᠨ ᠲᠠᠷᠤᠪᠴᠢ ᠢ ᠨᠢ ᠰᠢᠭᠤᠳ ᠳᠠᠷᠤᠨ᠎ᠠ᠃ ᠦᠨᠡᠰᠦᠨ ᠥᠩᠭᠡ ᠲᠡᠢ ᠶᠢ ᠨᠢ ᠤᠷᠳᠤ ᠳᠠᠷᠤᠭᠠᠳ ᠠᠪᠤᠨ᠎ᠠ᠃

2. ᠦᠰᠦᠭ ᠢᠯᠡᠷᠡᠳᠡᠭ ᠴᠣᠩᠬᠣᠨ ᠳᠤ ᠪᠤᠷᠤᠭᠤ ᠪᠢᠴᠢᠭᠰᠡᠨ ᠦᠰᠦᠭ ᠢᠶᠡᠨ ᠳᠠᠷᠤᠭᠠᠳ ᠪᠠᠢᠪᠠᠯ ᠤᠰᠠᠳᠬᠠᠵᠤ ᠪᠣᠯᠣᠨ᠎ᠠ᠃

3. ᠪᠠᠢᠨ᠎ᠠ᠂ ᠠᠬ᠎ᠠ᠂ ᠪᠠᠭ᠎ᠠ᠂ ᠴᠢᠮ᠎ᠡ᠂ ᠪᠠᠢᠯ᠎ᠠ᠂ ᠴᠢᠨ᠋ᠸ᠎ᠠ᠂ ᠭᠡᠬᠦ ᠦᠰᠦᠭ ᠦᠳ ᠢ ᠪᠢᠴᠢᠬᠦ ᠳᠦ (‍ᠨ᠎ᠠ ‍ᠯ᠎ᠠ ‍ᠭ᠎ᠠ) ᠤ ᠳᠠᠷᠤᠪᠴᠢ ᠢ ᠬᠡᠷᠡᠭᠯᠡᠨ᠎ᠡ᠃ 

ᠵᠢᠱᠢᠶᠡᠯᠡᠪᠡᠯ᠄ 
ᠪᠠᠢᠨ᠎ᠠ ︱ « ᠪ + ᠠ + ᠢ + ᠨ + (‍ᠨ᠎ᠠ ‍ᠯ᠎ᠠ ‍ᠭ᠎ᠠ) »
ᠪᠠᠭ᠎ᠠ ︱ « ᠪ + ᠠ + ᠭ + (‍ᠨ᠎ᠠ ‍ᠯ᠎ᠠ ‍ᠭ᠎ᠠ) »
ᠠᠮᠠ  ︱ « ᠠ + ᠮ + ᠠ »
ᠴᠢᠮ᠎ᠡ ︱ « ᠴ + ᠢ + ᠮ + (‍ᠨ᠎ᠠ ‍ᠯ᠎ᠠ ‍ᠭ᠎ᠠ) »
ᠪᠠᠢᠯ᠎ᠠ ︱ « ᠪ + ᠠ + ᠢ + ᠯ + (‍ᠨ᠎ᠠ ‍ᠯ᠎ᠠ ‍ᠭ᠎ᠠ) »

4. ᠶᠢᠨ ᠤᠨ ᠤ ᠭᠡᠵᠦ ᠳᠠᠭᠠᠪᠦᠷᠢ ᠤᠳ ᠢ ᠪᠦᠷ ( ᠶᠢ ᠳᠤ ᠤᠨ ) ᠲᠠᠷᠤᠪᠴᠢ ᠢ ᠬᠡᠷᠡᠭᠯᠡᠨ᠎ᠡ᠃

5. ᠵᠠᠷᠢᠮ ᠤᠨᠴᠠ ᠦᠰᠦᠭ ᠢ ᠪᠢᠴᠢᠬᠦ ᠳᠦ ᠳᠦᠷᠰᠦ ᠶᠢᠨ ᠰᠣᠩᠭᠤᠯᠳᠠ ᠳᠠᠷᠤᠪᠴᠢ ᠢ ᠬᠡᠷᠡᠭᠯᠡᠨ᠎ᠡ᠃ 

ᠵᠢᠱᠢᠶᠡᠯᠡᠪᠡᠯ᠄ 
ᠨᠠᠢ᠌ᠮᠠ ᠶᠢ ︱ « ᠨ + ᠠ + ᠢ + (ᠳᠦᠷᠰᠦ ᠶᠢᠨ ᠰᠣᠩᠭᠤᠯᠳᠠ ᠳᠠᠷᠤᠪᠴᠢ) + ᠮ + ᠠ »
ᠺᠣᠮᠫᠢᠦ᠋ᠲ᠋ᠧᠷ ︱ « ᠺ + ᠣ + ᠮ + ᠫ + ᠢ + ᠦ + (ᠳᠦᠷᠰᠦ ᠶᠢᠨ ᠰᠣᠩᠭᠤᠯᠳᠠ ᠳᠠᠷᠤᠪᠴᠢ) + ᠲ + (ᠳᠦᠷᠰᠦ ᠶᠢᠨ ᠰᠣᠩᠭᠤᠯᠳᠠ ᠳᠠᠷᠤᠪᠴᠢ) + ᠧ + ᠷ »

6. ᠦᠰᠦᠭ ᠦᠨ ᠢᠶᠡᠨ ᠠᠯᠢ ᠭᠡᠰᠡᠭ ᠢ ᠵᠠᠰᠠᠬᠤ ᠭᠡᠪᠡᠯ ᠬᠤᠷᠤᠭᠤ ᠪᠠᠷ ᠢᠶᠠᠨ ᠲᠡᠷᠡ ᠭᠠᠵᠠᠷ ᠨᠢ ᠳᠠᠷᠤᠭᠠᠳ ᠵᠠᠰᠠᠪᠠᠯ ᠪᠤᠯᠴᠢᠬᠠᠨ᠎ᠠ᠃
